# Reproductor WAVE WAVE Ver1.00
　Este es un software para reproducir datos PCM en formato .WAV de Windows en FX-DOS.<br><br>
Es compatible solo con archivos grabados en mono a 8/16 bits y 22050/44100 Hz. Tenga en cuenta que los archivos grabados con otras configuraciones no se reproducirán correctamente.<br><br>
※ Incluye muestras y listado de código fuente.
