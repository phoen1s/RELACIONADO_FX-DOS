# SEND Ver1.00 for Windows

Este es un software para Windows 95 y posteriores que permite transferir archivos desde un entorno Windows<br>
a la computadora de bolsillo CASIO «FX-890P».<br><br>
　Hasta ahora, para transferir archivos desde Windows al FX-890P, solo se podían usar software de terminal<br>
como HyperTerminal, pero esto presentaba varios inconvenientes: a menudo fallaba el control de flujo al<br>
transferir archivos grandes, solo se podían enviar archivos de texto, entre otros.<br><br>
　Por lo tanto, este software es una adaptación para Windows del programa de transferencia de archivos<br>
«SEND» que anteriormente había creado para PC-DOS.<br>
