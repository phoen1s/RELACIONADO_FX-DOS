# Font manager Ver1.00
　Este es un controlador de dispositivo para FX-DOS, creado para gestionar de manera integrada diversas fuentes, reduciendo la carga en cada aplicación y utilizando eficientemente los recursos de memoria.<br><br>
　Cabe señalar que este software por sí solo no tiene ninguna utilidad. Solo muestra su verdadero valor cuando se usa junto con otro software que utilice esta funcionalidad.<br>
Cuando ejecute software que utilice el gestor de fuentes, asegúrese de incluir este controlador en el archivo «CONFIG.SYS» de antemano.<br><br>

※ Para usar fuentes de caracteres kanji, se requiere por separado la fuente Erisa («ELISA.FNT»).<br><br>

※ Incluye listado de código fuente.
