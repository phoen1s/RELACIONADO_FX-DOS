# Graphic API manager Ver1.00
　Este es un controlador gráfico para FX-DOS que reúne funciones relacionadas con el procesamiento gráfico.<br><br>
　Al manejar gráficos a nivel de lenguaje ensamblador en la FX-890P, existen dos métodos: usar la BIOS o controlar directamente el controlador de LCD.<br>
El método que usa la BIOS permite dibujar gráficos fácilmente, pero su procesamiento es muy lento, lo que lo hace difícil de usar en situaciones que requieren alta velocidad, como en juegos.<br>
Por otro lado, el método de controlar directamente el controlador de LCD permite obtener una velocidad de dibujo rápida, pero para controlarlo se requiere un conocimiento detallado sobre el procesamiento de imágenes.<br><br>
　Por lo tanto, hemos creado un software que permite realizar dibujos en pantalla a alta velocidad con una facilidad similar a la de la BIOS.<br><br>

Cabe señalar que este software por sí solo no tiene ninguna utilidad. Solo muestra su verdadero valor cuando se usa junto con otro software que utilice esta funcionalidad.<br><br>

※ Este software utiliza internamente las funciones del gestor de fuentes «FONT.DRV», por lo que asegúrese de tener el gestor de fuentes preparado previamente.<br><br>
