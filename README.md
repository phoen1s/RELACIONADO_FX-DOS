# FX-DOS y aplicaciones para la Calculadora CASIO FX-890P/Z1

Este repositorio contiene una copia de seguridad de FX-DOS originalmente desarrollado y publicado por STEAR para la calculadora programable CASIO FX-890P (y modelos relacionados).

**Origen y Propósito:**  
Debido a que el sitio web oficial de STEAR (http://stear.c.ooco.jp/fx890p/indx.htm) ya no está disponible en línea, este repositorio se crea con el único propósito de preservar esta valiosa información técnica y evitar su pérdida.

El material incluye documentación, ejemplos, herramientas y posiblemente software relacionado con el uso y programación de esta calculadora clásica.

**Agradecimientos:**  
Un agradecimiento sincero y todo el crédito por el desarrollo, la compilación y la publicación original de este material es para STEAR. Su trabajo ha sido de gran ayuda para la comunidad de entusiastas de estas calculadoras.
