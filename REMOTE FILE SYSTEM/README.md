# Remote File System Ver0.76
Este es un controlador de dispositivo que funciona como el software cliente para la computadora de bolsillo, permitiendo el uso del software servidor «RSS» para PC, originalmente publicado por Koper para la serie PC-E500 como «RFS», en FX-DOS.<br><br>
　Al instalar esto en FX-DOS, será posible utilizar los discos de gran capacidad de una PC como unidades de expansión de FX-DOS.<br><br>
　Para obtener información detallada sobre RFS, consulte la documentación original de RFS for PC-E500.<br><br>

※ Para usar este software, se requieren FX-DOS y RSS por separado.<br><br>
