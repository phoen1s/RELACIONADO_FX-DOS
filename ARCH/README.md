# ARCH
　Al crear programas en lenguaje ensamblador, a menudo ocurre que se bloquean y uno tiene que realizar un "all reset" con pesar.<br>
　En esos momentos, una de las cosas que resulta sorprendentemente tediosa es la transferencia de archivos.<br><br>
　Por lo tanto, si se combinan múltiples archivos en uno solo en MS-DOS antes de transferirlos, sería un poco más fácil que transferir los archivos uno por uno, ¿verdad?<br><br>
　Así que ARCH (arco) es este programa que, ejecutándose en MS-DOS, crea un archivo compatible con las unidades de FX-DOS (disco RAM) a partir de múltiples archivos.<br><br>
　Si transfieres el archivo resultante a FX-DOS usando el comando copy, será equivalente a copiar toda la unidad.<br>
