# Nuevo Servidor de Sistema de Archivos Remoto 'NRS' para DOS/V Ver1.0
El «Sistema de Archivos Remoto RFS» (creado por Koper) es bastante útil, ya que permite usar una PC como una unidad virtual para la PC-E500 y manipular archivos de la PC desde la computadora de bolsillo.<br><br>
　RFS es compatible con la computadora X68000, pero NRS, que es este programa, permite su uso en DOS/V.<br><br>
　El archivo incluye solo el servidor para DOS/V, así que por favor prepare el RFS mencionado anteriormente por separado como controlador para la PC-E500.<br><br>
　Además, dado que el programa incluye internamente el controlador RS-232C para DOS/V, no es necesario preparar un controlador RS-232C por separado.<br><br>
